Steps to follow for Lab Prog-9:
1). Create following Directory Structure:
	C:\LabPrograms\No-9\
    ├── StudentDB.java
    ├── StudentDB.class (automatically generated after compilation)
    ├── sqlite-jdbc-3.50.3.0.jar (You have to download & Paste it in the same folder where .java file is created)
    |				  (link to download: https://github.com/xerial/sqlite-jdbc/releases)
    └── students.db (Created automatically when you run the program!)

	So, only one .java file you need to create & .jar file you need to download & paste it.

2). How to compile: Go to command prompt where your program Directory is :
	command to compile: javac StudentDB.java

3). How to Run : java -cp .;sqlite-jdbc-3.50.3.0.jar StudentDB
